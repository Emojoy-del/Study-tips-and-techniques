## Pomodoro Techniques
**contributer -** *Team member 1*
The pomodoro techniques involves studying for 25 minutes followed by a 5 minute break

**Benefits**
- improve focus
- reduces burnout
- Help manage time effectively
  
### Active recall
**contributer** *Team number 2* 
Active recall involves testing yourself instead of reading notes 

**Benefits**
- Strengthens memory
- Improve long term retention
- Help identify weak areas

#### Spaced repetition
**Contributer** *Team member 3*
Spaced repitition involves reviewing material at increasing intervals over time

**Benefits**
- Improve long term memory
- Reduce forgetting
- Efficient for exam preparations

**Teach someone else** 
If you can explain a topic clearly you understand it well

**Study with purpose**
1. what do i want to understand today 
2. what questions should i answer 
3. have a small goal for each session

**Write short notes** 
> Key point and highlight formulas or definitions
